# FEA - Food Expiration Application

## Overview

FEA (Food Expiration Application) is a web-based platform created to support initiatives against food waste. The system allows companies and organizations to participate in a food redistribution network, helping reduce unnecessary disposal of food while contributing to social and environmental sustainability.

This project was developed as part of an academic initiative and served as a practical experience in web application development, UI design, database integration, and software engineering principles.

---

## Features

- Partner registration system
- Responsive web interface
- Input validation for business information
- Food waste awareness platform
- Sustainability-focused solution
- Form validation with JavaScript
- ASP.NET Razor Pages architecture

## Main Objectives

The primary goals of this project are:

- Reduce food waste.
- Encourage sustainable practices.
- Connect businesses with redistribution initiatives.
- Raise awareness about food insecurity.
- Provide a simple platform for partner registration.

---

## Automated Testing

The project includes automated browser testing using Selenium WebDriver.

Test scenarios include:

- Navigation between pages.
- Form interaction and validation.
- User input simulation.
- Error handling validation.
- Alert message verification.

Example tested fields:

- Company Name
- Email
- Phone Number
- CNPJ
- ZIP Code (CEP)
- Partnership Motivation
